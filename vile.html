Installation and Usage <<Preview

Let's start.

1. install it on your html view

<script src="[your path]/vile.js"></script>
or load it from github
<script src="http://kleyverin.github.io/script/vile.js"></script>

2. Make a script on your HTML file or load a .js file. Inside it, make an object variable. We prefer to use 'var page'

var page = {}

3. Run the vile on top of the 'page' variable

Vile.initialize(page)
Now the 'page' variable will recognize every HTML tag which has the attribute 'vile-weave', e.g.
HTML:
<input name='username' vile-weave="inputUsername">
The 'page' variable recognize the input element above as:
page.inputUsername

The bound object now can be manipulated like any other elements.

page.inputUsername.value = "Vile.js"
Note: if you install JQuery. The element will instead turn into JQuery Element.
You should do this instead.
page.inputUsername.val("Vile.js")
